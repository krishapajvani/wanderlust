# Staycation (Wanderlust) ✈️🏠

Staycation is a full-stack MERN web application inspired by Airbnb. It allows users to explore, list, and book unique accommodations worldwide. This project was developed as part of a comprehensive learning journey at **Apna College**.

## 🚀 Live Demo
Check out the live project here: [Staycation - Live on Render](https://wanderlust-74o5.onrender.com/listings)

## ✨ Features
- **User Authentication:** Secure Sign-up and Login functionality.
- **Listing Management:** Users can create, edit, and delete their own property listings.
- **Search & Categories:** Filter stays by categories like Trending, Rooms, Iconic Cities, Mountains, Castles, Amazing Pools, and more.
- **Interactive Maps:** Visualized location data for every stay.
- **Review System:** Users can leave ratings and comments on listings.
- **Pricing Breakdown:** Automatic GST calculation (18%) and total price display.
- **Responsive Design:** Optimized for mobile, tablet, and desktop views.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Bootstrap), JavaScript, EJS (Embedded JavaScript)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Security:** Passport.js for authentication
- **Cloud Services:** Cloudinary (Image storage), Mapbox (Mapping services)
- **Deployment:** Render

## 💻 Installation
1. Clone the repository: `git clone https://github.com/krishapajvani/Wanderlust.git`
2. Install dependencies: `npm install`
3. Set up your `.env` file with MongoDB and Cloudinary credentials.
4. Run the application: `node app.js`
